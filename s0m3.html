Saison 0 · Mission 3 — S’outiller
Construire une “trousse” minimale commune (pour vous, les partenaires, les équipes et les publics) : des outils simples, low-tech, réutilisables, qui rendent l’action plus facile que l’inaction.

Mission 1 a produit une trace. Mission 2 a produit un artefact visible sans exposer. Mission 3 fabrique l’infrastructure humaine du projet : des outils de base pour que tout le monde puisse participer sans dépendre d’un expert, ni d’une plateforme.

Idée simple : un dispositif n’est éthique que s’il est reproductible. Ici, s’outiller signifie : rendre la participation accessible, douce, et soutenable — avec des outils qui fonctionnent même sans budget.

La trousse “Publics” (simple, adulte, non infantilisante)

20 min Sans compte Low-tech
  • Un carnet de traces (papier ou téléphone) : 5 lignes quand on veut
  • Un code simple : pas de noms, pas de visages, pas de détails intimes
  • Une question-guide : “Qu’est-ce qui m’a rendu un peu plus capable aujourd’hui ?”
  • Une passerelle : “Quelle suite douce serait possible ?”
  • Un droit : observer / se retirer / revenir plus tard

Micro-outil (à emporter)

Copiez ce mini-format et gardez-le quelque part (notes, carnet, bloc-notes). Il suffit pour jouer.

TRACE (5 lignes max) - Activité / moment : - Ce qui a compté : - Ce que j’ai appris sans que ce soit “une formation” : - Une suite douce possible : - Une question à poser ailleurs (optionnel) :

Générateur “TRACE PUBLIC” (optionnel)

Cet outil ne collecte rien. Tout reste sur votre appareil. Il génère une trace courte, copiable, sans infos personnelles.

Réussite : si vous avez une trace qui vous ressemble, sans vous exposer, vous êtes outillé·e.

La trousse “Équipes / lieux” (rituel de 5 minutes)

5 min Fin d’activité Sans charge

Le but n’est pas d’ajouter du travail. Le but est de rendre transmissible ce qui se passe déjà.

  • Rituel “fin d’activité” : 1 trace + 1 passerelle
  • Règle : anonymisation par défaut
  • Sortie : un petit “log” interne (copiable)
  • Option : un artefact sans exposition (Mission 2)

Mini-format d’équipe

LOG D’ÉQUIPE (5 min) - Activité : - Ce qui a renforcé le lien : - Un apprentissage observé : - Un “après” possible (suite douce) : - Une précaution éthique (si besoin) :

Générateur “LOG ÉQUIPE” + checklist éthique

Checklist express
✅ Anonyme par défaut ✅ Pas de visages / noms sans accord ✅ Publication optionnelle ✅ Si doute → interne

La trousse “Coordination projet” (sans contrôle)

30–60 min Réutilisable Minimal

Ici, s’outiller signifie : ne pas devenir un centre de pouvoir. On fabrique des outils pour distribuer l’autonomie.

  • Un dictionnaire commun (réel ↔ symbolique) : pour éviter les malentendus
  • Une charte 1 page (déjà prête) : pour rassurer et cadrer
  • Un format standard de mission (comme M1/M2) : reproductible
  • Un protocole de sortie : “on peut arrêter sans drame”
  • Une méthode de mémoire : documenter sans profiler

Protocole de sortie (copiable)

PROTOCOLE DE SORTIE - Un lieu peut mettre en pause à tout moment - Sans justification, sans dette symbolique - On garde ce qui est déjà produit (si accord) - On clôture par une trace simple : “ce qu’on retient”

Générateur “KIT LIEU” (1 page)

Crée une fiche de démarrage pour un lieu partenaire : cadre + contact + rythme + limites + besoins. Rien n’est envoyé automatiquement.

Réussite : si le lieu peut expliquer le pilote en 30 secondes et dire “stop” sans gêne, le dispositif est sain.

MISSION 3 — CONDITION DE RÉUSSITE

Mission 3 est réussie quand au moins une personne (public ou équipe) repart avec un outil simple qui lui donne plus de prise sur le réel, et quand au moins un lieu peut participer sans dépendre d’une expertise externe.

FERMETURE DE BOUCLE

Au début, il y avait une intention : augmenter les compétences et le pouvoir d’agir par le jeu, sans manipulation, en partant des lieux de liens. Mission 3 transforme cette intention en infrastructure : des outils communs, sobres, transmissibles. C’est le moment où le projet devient vraiment reproductible.
